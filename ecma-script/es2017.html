Array Includes

The includes() function on every array check if an array contains a specified value.

Return: boolean

    
        const cities = [ 'Lisbon', 'Porto', 'Roma', 'Paris'];
        console.log(cities.includes("Porto")) // true
        console.log(cities.includes("New York")) // false

Async functions
